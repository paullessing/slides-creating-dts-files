What are .d.ts files?

  • TypeScript definition files
    • C header files (.h)
  • Interface with JavaScript files
  • Type Checking

Why do I care?

  • Legacy code
  • Migration
  • Third-party Libraries
  • Intellisense
  • Publishing on npm
Never publish final code as Typescript on npm - other people will not be able to compile it.

Getting Started

  • tsc -d

Concepts

  • Type
  • Value
  • Namespace

Types

  • type T = number | string
  • interface I { v: T }
  • class C {}
  • enum E { A, B, C }
  • import { T } from 'm' // when T is a type
Things that are interfaces, compiled out by TypeScript

Values

  • const foo = 'bar' // also let and var
  • function hello()
  • enum E { A, B, C }
  • class C {}
  • namespace n { const x } // containing a value
  • module m { const x } // containing a value
  • import { v } from 'm' // when v is a value
Things that stay behind when you compile your code

Namespaces

  • namespace n { interface I { v: T } }
    • const x: A.B.C
    • const x = A.B.x

Value/Type: Class and Enum

  • Type
    • Interface of the class or enum
  • Value
    • Class constructor function
    • Enum object
